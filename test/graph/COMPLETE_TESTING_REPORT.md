## СЛЕДУЮЩИЕ ШАГИ

### Пункт 5: Fix remaining failing test
⏭️ **Осталось:** 1 тест (runId uniqueness в SubGraphNode)

**Проблема:** При множественных вызовах SubGraphNode генерируется одинаковый runId
```dart
final runId = '${context.runId}_sub_$id'; // Всегда одинаковый
```

**Решение:**
```dart
// Вариант 1: Добавить timestamp
final runId = '${context.runId}_sub_${id}_${DateTime.now().millisecondsSinceEpoch}';

// Вариант 2: Добавить счётчик
static int _counter = 0;
final runId = '${context.runId}_sub_${id}_${_counter++}';

// Вариант 3: Использовать UUID
final runId = '${context.runId}_sub_${id}_${Uuid().v4()}';
```

**Статус:** Не критично для текущей функциональности, но нужно исправить для production
